Interim Ad Landing Pages

README

Why are we creating static pages in Cascade Assets?

Ad Landing Pages are currently under development as a Cascade template. Click here to see the sample.

Because some ad campaigns were launched prematurely before the Cascade template could be completed, it was necessary to expedite release of certain pages.

These pages are included here in this special static directory for rapid development within the Cascade Assets repository. They will be released to the Cascade www2 server as they are completed.

Sample Static Template

This template should be finalized as much as possible before proceeding to develop new program pages using the workflow below.

Workflow

  2. In directory /landing-pages/2016-2017, create a directory for the new landing page, (e.g. /landing-pages/2016-2017/computational-science).
  3. Copy the contents of directory /landing-pages/2016-2017/sample into the new directory.
  4. Update any paths in _assets/master.self-2720c039bd36b71bb46be82be3386fc78d2b1cb0da0a8936cc65c21d62438b7b.css to use the full relative base page.
