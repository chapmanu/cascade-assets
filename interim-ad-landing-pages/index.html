Interim Static Ad Landing Page

README

Why?

Ad Landing Pages are currently under development as a Cascade template. See the sample here.

Because some ad campaigns were launched prematurely before the Cascade template could be completed, it was necessary to expedite release of certain pages.

These pages are included here in this special static directory for rapid development within the Cascade Assets repository. They will be released to the Cascade www2 server as they are completed.

Base Static Template

Click here for the base static template.

This template should be finalized as much as possible before proceeding to develop new program pages using the workflow below.

Workflow

  1. Pull latest version of branch interim-ad-landing-page from Github.
  2. In directory, create directory for new landing page, (e.g. executive-mba).
  3. Copy file _base.html and directory _assets with all its contents into new directory.
  4. In new directory, rename _base.html to index.html.
  5. Update index.html as desired. Review changes on local dev server.
  6. Remember to commit and push your changes to the interim-ad-landing-page branch
  7. When updates complete, deploy files to Cascade Assets www2 server.
